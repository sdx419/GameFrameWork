# 标题
使用 **"#"** 来作为标题的标记，**"#"** 的数量从1到6，分别表示1级标题到6级标题。

# 文本格式
## 字体
### 粗体
使用两个==星号**==或者两个下划线 __ 包围文字，可实现粗体效果。

例如：**粗体** 或者 __粗体__。

### 斜体
使用一个星号 * 或者一个下划线 _ 包围文字，可实现斜体。语法参考粗体实现部分。
### 粗斜体
使用三个星号*** 或者三个下划线_ 包围文字，可实现粗斜体，语法参考粗体实现部分。
## 分割线
使用三个以上的==星号*==或者减号- 或者 下划线_ 可实现分割线。星号直接可以有一个活多个空格，但是同一行不允许存在其他字符。
例如：
** **
___
-    -   -
## 删除线
使用两个波浪号 ~~ 包围文字，可实现删除效果。例如 ~~删除~~
## 下划线
使用\<u>和\</u> 包围文字，可实现下划线效果。  
例如：<u>下划线</u>。

## 脚注
脚注是对文本补充说明，可以使用如下格式来注明文本：
\[^脚注名称]
\[^脚注名称]:脚注内容
例如：煮酒论英雄[^说明]。温酒斩华佗[^故事详情]

[^说明]:煮酒论英雄是三国时期曹操和刘备的故事。
[^故事详情]:无此典故。
## 行内代码标记
使用符号\`包围代码，可实现代码效果。
例如：`pip install numpy`

当代码本身包含\`符号时，可使用两个\``符号来包围代码，进而实现代码效果。

例如：`` `code` ``的格式

## 文本高亮
文本高亮不是Markdown的语法，但是许多扩展支持。
使用两个等号== 包围文字可实现高亮效果。
例如 ==高亮==

# 列表
## 无序列表
使用"-"加空格作为开头，可以实现列表，例如：
 - 条目1
 - 条目2
 - 条目3
 - 条目4



显示效果为：
 - 条目1
- 条目2
## 无序列表
  使用数字加"."可以实现有序列表。例如：
1. 条目1
2. 条目2

## 列表的嵌套
可以实现有序列表和无序列表的混合嵌套，子列表需要缩进2-4个空格，例如：

- 条目1
  - 条目1-1
    1. 有序条目1-1-1
    2. 有序条目1-1-2
  - 条目1-2
    - 无序条目1-2-1
    - 无序条目1-2-3
- 条目2
- 条目3

## 复选框列表
主要用来标记是否完成。其语法为"-"加空格加中括号。中括号中间为空格时，表示未完成。中间为"x"时，表示已完成。例如： 

- [ ] 未完成的任务
- [x] 已完成的任务

# 引用块
使用">"可以实现引用功能，用以突出重要信息。
- 单行引用直接在行首输入“>”加空格即可。
- 长引用只需要在第一行行首输入“>”加空格。
- 引用可以实现嵌套，例如二层引用就是使用两个“>”加上空格。

例如：
>外层引用、
外层引用的内容
>> 这是中间层引用
中间层引用的内容1
中间层引用的内容2
>>> 这是内层引用

==在引用的内部，也可以使用标题，列表，代码，链接，图片等多种功能==

# 代码
## 行内代码
使用反引号"\`" 包围代码，可以实现代码格式。例如：
`printf("hello world!")`

## 代码块
使用三个 \`\`\` 包围代码可以实现代码块，并且可以指定语言（也可以不指定）。例如：

``` Cpp
for(int i = 0; i < 5; i++)
{
	print(i);
}
```

# 链接
## 普通链接
链接的语法格式为：\[链接名字](https://www.baidu.com "可选的标题")
或者可以直接使用链接地址：\<https://www.baidu.com>

> - 需要注意的是，链接地址不要落下“https://”。
> - 如果设置了标题，在展示界面，将鼠标放在链接上时，可以显示出标题。

例如：
[百度](https://www.baidu.com)
[谷歌](https://www.google.com "谷歌一下")

## 参考链接
参考链接将使用和定义进行分离，先定义，再使用。有点类似于论文中的参考文献。其使用语法为：

一段文字\[链接文字][参考标签]
\[参考标签]: 链接地址 "可选标题"

> 参考链接的定义通常放在文档末尾。

> ==如果链接文字和参考标签相同时，第二个中括号中间的内容可以省略。==

参考链接举例：
> 我平时会用[谷歌这个引擎][谷歌]搜索东西。
> 我平时会用[百度]搜索东西。

[谷歌]: https://www.google.com "谷歌搜索"
[百度]: https://www.baidu.com (百度一下)

## 跳转链接
跳转链接主要用于同一文档内跳转。

### 跳转到标题
目前可以实现跳转到标题。其语法为：
\[链接文本](#标题名)

例如：
点击这里可以进行[跳转](#文本格式)

# 图片

可以在Markdown中插入图片。其语法如下：
1. 开头是一个感叹号“!”。
2. 接着是一个方括号，方括号内是图片的代替文字，即当图片无法正常显示时，可显示代替文字。
3. 接着是一个小括号，小括号内是图片路径，然后可以加上一个可选择的图片标题。

> **格式为：\![图片文字](图片路径， “图片标题”)**

其中，路径可以是相对路径、绝对路径和网址。

## 相对路径

使用相对路径时，通常将图片放在指定文件夹中，如./images。以便管理资源。
例如：**!\[测试图片](.images/test.png)**，其渲染效果如下：

![测试图片](./images/test.png)
## 绝对路径
图片路径替换为文件的绝对路径也可以实现同样的效果。
例如：**!\[测试图片](F:\MarkDown_Note\Markdown_Grammar\images\test.png)**
其渲染效果如下：
![测试图片](F:\MarkDown_Note\Markdown_Grammar\images\test.png)

## 网址
直接将图片路径替换成网络图片的网址，即可显示网络图片
例如：**!\[RUNOOB 图标](https://static.jyshare.com/images/runoob-logo.png)**
其渲染效果如下所示：

>![RUNOOB 图标](https://static.jyshare.com/images/runoob-logo.png)

### 图片和链接的组合

可以利用两者组合，实现点击图片访问指定网址的效果。其语法为：
\[!\[图片代替文字](图片路径)](待访问网址 "可选的标题")。具体来讲：

1. 首先是一个方括号，其内部为图片的表示方式，即由感叹号、中括号、中括号内的代替文字、小括号、小括号内的图片路径，
2. 接着是一个小括号，小括号内是待访问的网址，以及一个可选的标题。

例如：
[![点击此图片可访问百度](./images/test.png)](https://www.baidu.com "百度一下")
